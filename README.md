## Game Zone

**Presentado por:**
Geovanny Santiago Pino Avendaño

## Descripción

Game Zone es una aplicación desarrollada en Java que permite gestionar un catálogo de videojuegos físicos y digitales. El sistema incluye funcionalidades de registro, búsqueda, listado y venta de videojuegos, utilizando programación orientada a objetos, persistencia de datos en archivos JSON y una interfaz gráfica desarrollada con JavaFX.

El proyecto implementa conceptos como herencia, polimorfismo, abstracción, interfaces y arquitectura por capas para la administración eficiente de videojuegos y ventas.
